Hasura auto-generates (GraphQL and REST) APIs faster, while still using your databases’ native data types

Supported databases include:

  • PostgreSQL
  • SQL
  • Server
  • BigQuery
  • Yugabyte
  • DB Citus / Hyperscale

Bringing in external data sources

Hasura’s GraphQL engine enables you to join data across tables and add remote data sources via remote relationships (AKA remote joins). This is great for bringing third-party APIs from Stripe, Spotify, or Auth0. The remote data source can either be a remote schema or the type returned from an action.
